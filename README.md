### 🤖 Fiedler AI *(In Development)*
> AI-powered study guidance platform for Brazilian public service exam candidates targeting auditor positions (CGU, TCU, TCE, SEFAZ).

- Personalized study planning via **Anthropic Claude API**
- Full SaaS with auth (Clerk), payments (Stripe), and database (Supabase + Prisma)
- Stack: **Next.js 15 · TypeScript · Supabase · Prisma · Stripe · Anthropic API**
- Tiered pricing: Free / Pro (R$39/mo) / Elite (R$79/mo)

### 🗄️ PostgreSQL 16 + pgBackRest — Production Backup Infrastructure
> Designed and deployed a robust backup solution for a RHEL 8.8 production server at Embrapa.

- pgBackRest on a dedicated Rocky Linux VM with full, differential, and incremental strategies
- Navigated cross-subnet infrastructure constraints with a **PowerShell + SSH key auth** workaround
- Automated integrity verification and transfer monitoring
- Full technical documentation: installation guide, situation report, migration action plan

- Deployed individual Google Sheets per department with IMPORTRANGE integration and cell protection
- Looker Studio dashboard consumed by department heads and senior management
